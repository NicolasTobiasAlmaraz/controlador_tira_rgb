Descripción detallada

API de pulsadores.

Autor
Nicolás Almaraz
Versión
rev 1.0
Fecha
28/06/2024

Esta API se encarga de leer los tres pulsadores eliminando los rebotes y glitches de cada uno de ellos. Para ello cuenta con una función de "lectura validada": estado_pulsadores_t pulsadoresApi_getPulsadores();

Cabe destacar que por hardware los pulsadores en estado IDLE hacen que la placa reciba 0V y cuando son presionados 3.3V
